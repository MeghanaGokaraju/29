
#  **StepCount Pro — README**

##  Overview

**StepCount Pro** is an interactive fitness web application designed to track users’ daily step counts, visualize progress toward personalized goals, and display weekly trends. The application emphasizes clean UI, meaningful analytics, progress visualizations, and motivational elements that encourage users to stay active.

This single-page web app is built using **HTML, CSS, and JavaScript** with no external dependencies, making it easy to run in any browser.

---

## **Features / Functional Requirements**

### **1. Daily Activity Tracking**

* Displays daily steps using mock data.
* Shows goal progress using a dynamic animated progress bar.
* Allows users to manually set or update daily step goals.

### **2. Visual Progress Analytics**

* Weekly steps visualized with a simple custom bar chart (Canvas API).
* Color-coded indicators and smooth animations included.

### **3. Weekly / Monthly Summaries**

* Weekly total and average analyzed to show performance insights.
* Trend message helps users understand their progress relative to weekly average.

### **4. Goal Management**

* Users can update daily goals anytime.
* Progress visual updates immediately after changes.

### **5. Fully Responsive UI**

* Optimized for both mobile and desktop displays.
* Smooth scaling and layout adjustments.

---


###  **Achievement Badges**

* Awards badges such as:

  * **5,000 Steps Badge**
  * **10,000 Steps Champion**
  * **15,000 Steps Beast**
* Fully animated pop-in effect.

###  **Trend Analytics**

* Displays whether today's steps are above or below weekly average.
* Helps users understand long-term patterns.

###  **Motivational Messages**

* Randomized encouragement appears daily to boost engagement.

###  **Dark/Light Mode**

* Clean theme toggle button.
* Smooth transition animations.
* Fully responsive theme variables.

###  **Animated Visuals**

* Progress bar animation.
* Badge pop animation.
* Weekly chart bars grow dynamically.
